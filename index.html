This Website Is To Show My Histogram

The histogram was made using the bookworm penguin

The following histogram shows the distribution of the bookworm penguin's quiz grades over a forty day period.

The histogram was made using excel and the data provided by the moodle page. I entered the data into excel, went to the insert tab and pressed add histogram. I then had to add axes to the histogram, a title, and reorder the bins used to separate the data. With this and a color change, I got the graph shown above.
